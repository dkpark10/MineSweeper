# 버튼 클릭에 따른 이벤트

### 왼클릭

1. 지뢰여부는 깃발이 있는지 없는지 확인한다음 지뢰를 체크한다.
2. 주위 지뢰가 하나도 없는 청정한 셀일경우 연쇄충돌을 일으키고 버튼을 set disable 한다.
3. 지뢰가 없지만 주위 지뢰가 하나라도 있는경우 연쇄충돌을 일으키지도 않고 disable을 set하지 아니한다. 
4. 깃발 클릭은 아무런 이벤트를 일으키지 아니한다. 
5. disable 버튼을 클릭시 아무런 이벤트를 일으키지 아니한다.

### 휠클릭

1. 주위 깃발 위치,갯수와 지뢰 갯수,위치가 동일하다면 연쇄충돌을 일으킨다.
2. 주위 깃발 위치,갯수와 지뢰 갯수,위치가 하나라도 틀리다면 게임을 끝낸다. 

### 오른클릭

1. 방문된곳이라면 아무런 이벤트를 발생하지 아니한다.
2. 깃발이 없다면 깃발을 set한다.
3. 깃발이 있다면 깃발을 unset한다.
